> features What makes it tick

Multi-language

Parses and beautifies traces from Node.js, Python, and Java — all with the same command.

Pipe & File Input

Works three ways: pipe a trace, npx github:kvnlnk/stacylize file.txt, or paste directly into the terminal prompt.

Zero Config

No setup file, no API key, no dependencies to install globally. Just npx github:kvnlnk/stacylize and you’re done.

Customizable

Choose from built-in color themes or define your own palette with a simple --theme flag.

> install Get started in seconds

terminal — bash 
# Run directly (no install needed)
$ npx github:kvnlnk/stacylize_

# Or install globally
$ npm install -g github:kvnlnk/stacylize

Usage examples:

# Pipe a trace from a process $ node app.js 2>&1 | npx github:kvnlnk/stacylize
# Pass a file $ npx github:kvnlnk/stacylize trace.txt
# Paste mode (interactive) $ npx github:kvnlnk/stacylize --paste
> stacylize beautify stack traces
